# Pentab Teaching Tool

A standalone HTML-based teaching tool with a built-in no-code slide builder 
and integrated pentab drawing canvas — runs directly in the browser with 
no external dependencies.

## Features
- No-code slide builder
- Smooth, lag-free freehand drawing using pointer and coalesced events
- Pen and eraser tools
- Customizable colors and templates
- Works fully offline, downloadable and usable locally

## Tech Stack
HTML, JavaScript
